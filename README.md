## Introduction

Android SDK for [Computop](https://www.computop.com/).


## Add the SDK to Your Project

    implementation 'com.computop.sdk:lib:<VERSION>'

## How to build

    gradlew installDebug

#### Build Requirements

- JDK7, JDK8
- Android Build Tools 29.0.3
- Android SDK 29
- Android min SDK API 16
